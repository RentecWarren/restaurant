Savvior Demo - Command Line Restaurant 
===============

- Small business use case  
- Doctrine ORM 
- Test Driven Development

## Installation

```bash
$ composer install
```

## Setup

Crete DB

```bash
$ php bin/doctrine orm:schema-tool:create
```

## Execute commands

### List available commands

```bash
$ php bin/restaurant-cli 
```
